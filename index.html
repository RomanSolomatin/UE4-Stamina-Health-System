Unreal Engine 4 - Stamina and Health System (Blueprints)

Download this project as a .zip file Download this project as a tar.gz file

This is my first experience with Unreal Engine 4 (4.10). I wanted to build a complete stamina and health system for an FPS game.

Blueprints for heads-up displays (HUD)

  • Stamina HUD
  • Health HUD
  • Blood Marks HUD

Blueprints for systems

  • Stamina system
  • Health system
  • Character Movement system

Blueprints for menus

  • DeathMenu (used to respawn when you die)

How to use the systems

Initialize the systems within a character's EventGraph Blueprint

Run

Jump

Aim

Tick

Inflict Damage
